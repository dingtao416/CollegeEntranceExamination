3. 已知集合 A = {sin(7π/6)，cos(5π/3)，tan(5π/4)}，B = {√3/2，-1/2，1}，则 A ∩ B =

4. 曲线 y = 5x + 8lnx 在点 (1,5) 的切线方程为

A. y = 3x + 2
B. y = 5x

答案：D

6. 已知函数 f(x) = (x + 2) / (eˣ + a) 的最大值为 1，则 a =

A. 1/2　　B. 1　　C. 3/2　　D. 2

答案：B

解析：设最大值在 x₀ 处取得，则 f(x₀) = 1 且 f'(x₀) = 0。

由 f(x₀) = 1 得 x₀ + 2 = eˣ⁰ + a；由 f'(x₀) = 0 得 a = (x₀ + 1)eˣ⁰。

联立得 x₀ + 2 = (x₀ + 2)eˣ⁰。若 x₀ = -2，则 a < 0，不合选项；故 x₀ = 0，a = 1。

10. 在空间中，A，B 为两个定点，动点 C 到直线 AB 的距离为 2，动点 D 到直线 AB 的距离为 1，二面角 C - AB - D 为 60°，则

A. ∠CAD ≥ 60°
B. CD ≥ √3
C. 当 AB ⟂ CD 时，CD ⟂ 平面 ABD
D. 当 AB ⟂ 平面 ACD 时，AC ⟂ AD

答案：BC

解析：以 AB 为轴，C、D 到 AB 的垂直距离分别为 2 和 1，且两垂面夹角为 60°。

CD² 等于沿 AB 方向的差的平方加上垂直截面内的距离平方，故 CD² ≥ 2² + 1² - 2·2·1·cos60° = 3，B 正确。

若 AB ⟂ CD，则 C、D 在 AB 方向投影相同，此时 CD 同时垂直 AB 和 AD，故 CD ⟂ 平面 ABD，C 正确。A、D 可由改变点在 AB 方向的位置构造反例。

11. 已知圆 C₁：(x + 1)² + y² = 1，C₂：(x - 1)² + y² = 1，C₃：x² + (y - √3)² = 1，直线 l：y = kx + b 与 C₁，C₂，C₃ 均有两个交点，记 l 被 C₁，C₂，C₃ 截得的弦长分别为 s₁，s₂，s₃，则

A. k 可以取任意实数
B. 满足 s₁ = s₂ = s₃ 的直线 l 共有 3 条
C. 满足 s₁ + s₂ + s₃ = 3 的直线 l 多于 3 条
D. 当 b = 0 时，s₁ + s₂ + s₃ 的最大值为 2√21 / 3

答案：BCD

解析：圆心分别为 O₁(-1,0)，O₂(1,0)，O₃(0,√3)。直线 l 到三圆圆心的距离分别为 |b - k|/√(1 + k²)，|b + k|/√(1 + k²)，|b - √3|/√(1 + k²)。

若 k = 1/√3，则 O₂、O₃ 在直线法向上的投影距离刚好为 2，无法让 l 与这两个单位圆都相交于两个点，故 A 错。

由弦长公式 s = 2√(1 - d²) 可知 s₁ = s₂ = s₃ 等价于三圆心到 l 的距离相等，解得 3 条直线，B 正确。

取 k = 0 时，s₁ + s₂ + s₃ = 4√(1 - b²) + 2√(1 - (b - √3)²)，在允许区间内除 b = √3/2 外还存在另一个解；再加上 B 中的另外两条等弦长直线，满足 s₁ + s₂ + s₃ = 3 的直线多于 3 条，C 正确。

当 b = 0 时，令 t = 1/√(1 + k²)，则 s₁ + s₂ + s₃ = 4t + 2√(1 - 3t²)。求最大值得 2√21 / 3，D 正确。

13. 已知 f(x) = 2sin(ax + θ)（a ∈ Z，0 ≤ θ < 2π）是偶函数，f(x) 在区间 (0, π/2) 单调递增，则 θ = ______，f(2π/3) = ______。

答案：θ = 3π/2，f(2π/3) = 1

解析：f(x) 为偶函数，需 cosθ = 0，故 θ = π/2 或 3π/2。

若 θ = π/2，则 f(x) = 2cos ax，不满足题设递增；故 θ = 3π/2，此时 f(x) = -2cos ax。

在 (0,π/2) 上单调递增要求 |a| = 1 或 2。此时 cos(2aπ/3) = -1/2，所以 f(2π/3) = -2cos(2aπ/3) = 1。

14. 设实数 aᵢ 满足：存在数列 {aₙ}，使得对于任意 n ∈ N*，均有 a₁ + a₂ + … + a₃ₙ = n² + n。且 {aₙ} 中有某连续 9 项 aₖ，aₖ₊₁，…，aₖ₊₈ 是公比为 q 的等比数列，则 q 的最大值为 ______。

答案：∛(3/2)

解析：由 a₁ + a₂ + … + a₃ₙ = n² + n，可得每 3 项一组的和为 a₃ₙ₋₂ + a₃ₙ₋₁ + a₃ₙ = 2n。

连续 9 项等比数列中必含两个相邻的完整三项组。若首项对应的组三项和为 2r，则下一完整组三项和为 2(r + 1)，而等比数列中相隔 3 项的倍数关系为 q³。

因此 q³ = (r + 1)/r ≤ 3/2，且 r = 2 时可取到最大值，所以 q 的最大值为 ∛(3/2)。

已知函数 f(x) 的定义域为 R，且当 x < 0 时，f(x) = 2ˣ。对于任意 x₀ ∈ R，定义集合 D(x₀) = {d ∈ R | f(x₀ + d) > f(x₀)}。

（1）若当 x ≥ 0 时，f(x) = 1 - x，求 D(-1)；

（2）若 f(x) 是奇函数，f(x₁) ≤ f(x₂)，且 x₁x₂ ≠ 0，证明：D(x₂) ⊂ D(x₁)；

（3）设 f(x) 满足：① 若 f(x₁) ≤ f(x₂)，则 D(x₂) ⊂ D(x₁)；② 当 0 < x < 1 时，f(x) < f(0)。

（ii）证明：f(x) 在区间 [0,+∞) 上单调递增。

答案：（1）D(-1) = {d ∈ R | 0 < d < 3/2}；（2）结论成立；（3）（i）f(0) ≥ 1；（ii）f(x) 在 [0,+∞) 上单调递增。

参考解答：

（1）f(-1) = 2⁻¹ = 1/2。

D(-1) = {d ∈ R | f(-1 + d) > 1/2}。

令 t = -1 + d，解 f(t) > 1/2。

若 t < 0，则 f(t) = 2ᵗ，2ᵗ > 1/2 = 2⁻¹，所以 t > -1，即 -1 < t < 0。

若 t ≥ 0，则 f(t) = 1 - t，1 - t > 1/2，所以 0 ≤ t < 1/2。

综上，t ∈ (-1, 1/2)。

代入 t = -1 + d，得 0 < d < 3/2。

故 D(-1) = {d ∈ R | 0 < d < 3/2}。

（2）因为 f(x) 是奇函数，所以：

当 x < 0 时，f(x) = 2ˣ；当 x = 0 时，f(x) = 0；当 x > 0 时，f(x) = -2⁻ˣ。

即：

f(x) = 2ˣ，x < 0；f(x) = 0，x = 0；f(x) = -2⁻ˣ，x > 0。

当 x < 0 时，D(x) = (0, -x)。

当 x > 0 时，D(x) = (-∞, -x] ∪ (0, +∞)。

分类讨论：

① 若 x₁ < 0，x₂ < 0，由 f(x₁) ≤ f(x₂) 得 x₁ ≤ x₂ < 0，所以 D(x₂) ⊂ D(x₁)。

② 若 x₁ > 0，x₂ > 0，由 f(x₁) ≤ f(x₂) 得 0 < x₁ ≤ x₂，所以 D(x₂) ⊂ D(x₁)。

③ 若 x₁ > 0，x₂ < 0，则 f(x₁) < 0 < f(x₂)，满足 f(x₁) ≤ f(x₂)，此时 D(x₂) = (0, -x₂) ⊂ (0,+∞) ⊂ D(x₁)。

④ 若 x₁ < 0，x₂ > 0，则 f(x₁) > 0 > f(x₂)，与 f(x₁) ≤ f(x₂) 矛盾。

综上，D(x₂) ⊂ D(x₁)。

（3）（i）假设 f(0) < 1。

若 f(0) ≤ 0，取 x₁ = -1/2，则 f(x₁) = 2⁻¹ᐟ² > 0，所以 f(0) < f(x₁)。由条件①知 D(0) ⊃ D(x₁)。

取 d = 1/4，则 x₁ + d = -1/4 < 0，f(x₁ + d) = 2⁻¹ᐟ⁴ > 2⁻¹ᐟ² = f(x₁)，所以 d ∈ D(x₁)。

于是 d ∈ D(0)，即 f(d) > f(0)。

但 d = 1/4 ∈ (0,1)，由条件②知 f(d) < f(0)，矛盾。

若 0 < f(0) < 1，令 x₀ = log₂ f(0)，则 x₀ < 0，且 f(x₀) = f(0)。

由条件①可得 D(x₀) = D(0)。

取 d₀ = min{-x₀/2, 1/2}，则 0 < d₀ ≤ 1/2 < 1。

由条件②知 f(d₀) < f(0)，所以 d₀ ∉ D(0)。

又 x₀ + d₀ ≤ x₀/2 < 0，因此 f(x₀ + d₀) = 2ˣ⁰⁺ᵈ⁰ > 2ˣ⁰ = f(x₀)，所以 d₀ ∈ D(x₀)，矛盾。

综上，假设不成立，所以 f(0) ≥ 1。

（ii）先证：当 x > 0 时，f(x) ≤ 0。

若 x ∈ (0,1)，假设 f(x) > 0。取 c < 0，使得 f(c) = 2ᶜ < f(x)。

由条件②知 f(x) < f(0)，所以 f[x + (-x)] = f(0) > f(x)，即 -x ∈ D(x)。

又由条件①及 f(c) < f(x) 得 D(c) ⊃ D(x)，所以 -x ∈ D(c)。

于是 f(c - x) > f(c)。

但 c - x < c < 0，而当 x < 0 时 f(x) = 2ˣ 单调递增，所以 f(c - x) < f(c)，矛盾。

所以对任意 x ∈ (0,1)，f(x) ≤ 0。

若 x ≥ 1，假设 f(x) > 0。取 b < 0，使得 f(b) = 2ᵇ < f(x)。

由 f[b + (x - b)] = f(x) > f(b)，知 x - b ∈ D(b)。

令 b' = -1/2(x - b)。因为 x ≥ 1，b < 0，所以 x - b > 1，b' < -1/2 < 0。

可推出与已得的 0 < t < 1 时 f(t) ≤ 0 矛盾。

故对任意 x > 0，f(x) ≤ 0。

最后取 0 < x < y。

因为 y - x > 0，所以 -(y - x) - 1 < -1 < 0。

于是 f[-(y - x) - 1] = 2⁻⁽ʸ⁻ˣ⁾⁻¹ > 0。

又 x > 0，所以 f(x) ≤ 0，故 f(x) < f[-(y - x) - 1]。

由条件①知 D(x) ⊃ D[-(y - x) - 1]。

因为 f[-(y - x) - 1 + (y - x)] = f(-1) > f[-(y - x) - 1]，所以 y - x ∈ D[-(y - x) - 1]。

因此 y - x ∈ D(x)，于是 f[x + (y - x)] > f(x)，即 f(y) > f(x)。

所以 f(x) 在区间 [0,+∞) 上单调递增。
