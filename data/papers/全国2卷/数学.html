2025 年全国统一高考数学试卷（全国二卷）题目及解答

一、单选题：本题共 8 小题，每小题 5 分，共 40 分。在每小题给出的选项中，只有一项是符合题目要求的。

1. 样本数据 2，8，14，16，20 的平均数为（　）

A. 8　　B. 9　　C. 12　　D. 18

答案：C

解：平均数为 (2 + 8 + 14 + 16 + 20) / 5 = 12。

2. 已知 z = 1 + i，则 1 / (z - 1) =（　）

A. -i　　B. i　　C. -1　　D. 1

答案：A

解：z - 1 = i，所以 1 / (z - 1) = 1 / i = -i。

3. 已知集合 A = {-4, 0, 1, 2, 8}，B = {x | x³ = x}，则 A ∩ B =（　）

A. {0,1,2}　　B. {1,2,8}　　C. {2,8}　　D. {0,1}

答案：D

解：x³ = x，即 x(x - 1)(x + 1) = 0，B = {-1,0,1}，故 A ∩ B = {0,1}。

4. 不等式 (x - 4) / (x - 1) ≥ 2 的解集是（　）

A. {x | -2 ≤ x ≤ 1}
B. {x | x < -2}
C. {x | -2 ≤ x < 1}
D. {x | x > 1}

答案：C

解：(x - 4) / (x - 1) - 2 = -(x + 2) / (x - 1) ≥ 0，即 (x + 2) / (x - 1) ≤ 0，得 -2 ≤ x < 1。

5. 在 △ABC 中，BC = 2，AC = 1 + √3，AB = √6，则 A =（　）

A. 45°　　B. 60°　　C. 120°　　D. 135°

答案：A

解：由余弦定理，cos A = (AB² + AC² - BC²) / (2AB·AC) = √2 / 2，所以 A = 45°。

6. 设抛物线 C：y² = 2px（p > 0）的焦点为 F，点 A 在 C 上，过 A 作 C 的准线的垂线，垂足为 B，若直线 BF 的方程为 y = -2x + 2，则 |AF| =（　）

A. 3　　B. 4　　C. 5　　D. 6

答案：C

解：F = (p/2,0)，B = (-p/2,y_A)。直线 BF 的斜率为 -y_A / p = -2，且过 F 时截距为 p，所以 p = 2，y_A = 4。

由 y_A² = 2px_A 得 x_A = 4，F = (1,0)，故 |AF| = √((4 - 1)² + 4²) = 5。

7. 记 Sₙ 为等差数列 {aₙ} 的前 n 项和，若 S₃ = 6，S₅ = -5，则 S₆ =（　）

A. -20　　B. -15　　C. -10　　D. -5

答案：B

解：设首项为 a，公差为 d。由 S₃ = 3a + 3d = 6，S₅ = 5a + 10d = -5，得 a = 5，d = -3。故 S₆ = 3(2a + 5d) = -15。

8. 已知 0 < a < π，cos(a/2) = √5 / 5，则 sin(a - π/4) =（　）

A. √2 / 10　　B. √2 / 5　　C. 3√2 / 10　　D. 7√2 / 10

答案：D

解：因 0 < a/2 < π/2，sin(a/2) = 2√5 / 5。于是 sin a = 4/5，cos a = -3/5。

sin(a - π/4) = (sin a - cos a) / √2 = 7√2 / 10。

二、多选题：本题共 3 小题，共 18 分。在每小题给出的选项中，有多项符合题目要求。

9. 记 Sₙ 为等比数列 {aₙ} 的前 n 项和，q 为 {aₙ} 的公比，q > 0，若 S₃ = 7，a₃ = 1，则（　）

A. q = 1/2
B. a₅ = 1/9
C. S₅ = 8
D. aₙ + Sₙ = 8

答案：AD

解：设 t = 1/q，则 S₃ = a₃(t² + t + 1) = 7，得 t² + t - 6 = 0。因 t > 0，t = 2，q = 1/2。

此时 a₅ = 1/4，S₅ = 31/4，且 Sₙ = 8(1 - 2⁻ⁿ)，aₙ = 8·2⁻ⁿ，所以 aₙ + Sₙ = 8。

10. 已知 f(x) 是定义在 R 上的奇函数，且当 x > 0 时，f(x) = (x² - 3)eˣ + 2，则（　）

A. f(0) = 0
B. 当 x < 0 时，f(x) = -(x² - 3)e⁻ˣ - 2
C. f(x) = 2 当且仅当 x = √3
D. x = -1 是 f(x) 的极大值点

答案：ABD

解：奇函数在 0 处有定义，故 f(0) = 0。x < 0 时，f(x) = -f(-x) = -(x² - 3)e⁻ˣ - 2。

在 x < 0 时也可能有 f(x) = 2，故 C 错。对 x < 0，f'(x) = e⁻ˣ(x - 3)(x + 1)，所以 x = -1 左增右减，是极大值点。

11. 双曲线 C：x²/a² - y²/b² = 1（a > 0，b > 0）的左、右焦点分别是 F₁、F₂，左、右顶点分别为 A₁、A₂，以 F₁F₂ 为直径的圆与 C 的一条渐近线交于 M、N 两点，且 ∠NA₁M = 5π/6，则（　）

A. ∠A₁MA₂ = π/6
B. |MA₁| = 2|MA₂|
C. C 的离心率为 √13
D. 当 a = √2 时，四边形 NA₁MA₂ 的面积为 8√3

答案：ACD

解：取渐近线 y = (b/a)x，则圆 x² + y² = c² 与它交于 M(a,b)，N(-a,-b)，其中 c² = a² + b²。

由 ∠NA₁M = 5π/6 得 -b / √(4a² + b²) = -√3 / 2，故 b² = 12a²，离心率 e = c/a = √13。

此时 ∠A₁MA₂ = π/6，且 |MA₁| = 4a，|MA₂| = 2√3a，B 错。若 a = √2，则四边形面积为 2ab = 8√3。

三、填空题：本题共 3 小题，每小题 5 分，共 15 分。

12. 已知平面向量 a = (x,1)，b = (x - 1,2x)，若 a ⟂ (a - b)，则 |a| = ______。

答案：√2

解：a - b = (1,1 - 2x)。由 a·(a - b) = 0 得 x + 1 - 2x = 0，x = 1，所以 |a| = √(1² + 1²) = √2。

13. 若 x = 2 是函数 f(x) = (x - 1)(x - 2)(x - a) 的极值点，则 f(0) = ______。

答案：-4

解：f'(2) = 0。展开 f(x) = x³ - (a + 3)x² + (3a + 2)x - 2a，故 f'(2) = 2 - a，得 a = 2。于是 f(0) = -4。

14. 一个底面半径为 4cm，高为 9cm 的封闭圆柱形容器（容器壁厚度忽略不计）内有两个半径相等的铁球，则铁球半径的最大值为 ______ cm。

答案：2.5

解：设球半径为 r，则球心在半径 4 - r、高 9 - 2r 的圆柱内。两球心距离最大为 √[4(4 - r)² + (9 - 2r)²]。

令该最大距离等于 2r，得 4r² - 68r + 145 = 0，取可行根 r = 5/2。

四、解答题：本题共 5 小题，共 77 分。解答应写出文字说明、证明过程或演算步骤。

15.（13 分）

已知函数 f(x) = cos(2x + φ)（0 < φ < π），f(0) = 1/2。

（1）求 φ；

（2）设函数 g(x) = f(x) + f(x - π/6)，求 g(x) 的值域和单调区间。

答案：（1）φ = π/3；（2）值域 [-√3, √3]；增区间 [-7π/12 + kπ, -π/12 + kπ]，减区间 [-π/12 + kπ, 5π/12 + kπ]，k ∈ Z。

解：（1）由 f(0) = cos φ = 1/2 且 0 < φ < π，得 φ = π/3。

（2）g(x) = cos(2x + π/3) + cos 2x = √3 cos(2x + π/6)，所以值域为 [-√3, √3]。

g'(x) = -2√3 sin(2x + π/6)。当 2x + π/6 ∈ [π + 2kπ, 2π + 2kπ] 时单调递增，即 x ∈ [-7π/12 + kπ, -π/12 + kπ]。

当 2x + π/6 ∈ [2kπ, π + 2kπ] 时单调递减，即 x ∈ [-π/12 + kπ, 5π/12 + kπ]。

16.（15 分）

已知椭圆 C：x²/a² + y²/b² = 1（a > b > 0）的离心率为 √2 / 2，长轴长为 4。

（1）求 C 的方程；

（2）过点 (0,-2) 的直线 l 与 C 交于 A、B 两点，O 为坐标原点，若 △OAB 的面积为 √2，求 |AB|。

答案：（1）x²/4 + y²/2 = 1；（2）|AB| = √5。

解：（1）由长轴长为 4，得 a = 2。又 e = c/a = √2/2，故 c = √2，b² = a² - c² = 2，所以 C：x²/4 + y²/2 = 1。

（2）设 l：y = mx - 2。代入椭圆得 (1 + 2m²)x² - 8mx + 4 = 0。

设 A、B 横坐标为 x₁、x₂，则 S△OAB = |x₂ - x₁|。由面积为 √2，得 |x₂ - x₁| = √2。

又 |x₂ - x₁| = 4√(2m² - 1) / (1 + 2m²)，解得 m² = 3/2。故 |AB| = √(1 + m²)|x₂ - x₁| = √5。

17.（15 分）

如图，在四边形 ABCD 中，AB ∥ CD，∠DAB = 90°，F 为 CD 的中点，点 E 在 AB 上，EF ∥ AD，AB = 3AD，CD = 2AD，将四边形 EFDA 沿 EF 翻折至四边形 EFD'A'，使得面 EFD'A' 与面 EFCB 所成的二面角为 60°。

（1）证明：A'B ∥ 平面 CD'F；

（2）求面 BCD' 与面 EFD'A' 所成的二面角的正弦值。

答案：（1）A'B ∥ 平面 CD'F；（2）正弦值为 2√5 / 5。

解：不妨设 AD = 1，取坐标 A(0,0,0)，B(3,0,0)，D(0,1,0)，C(2,1,0)。则 F(1,1,0)，E(1,0,0)。

沿 EF 翻折 60° 后，可取 A'(1/2,0,√3/2)，D'(1/2,1,√3/2)。

（1）平面 CD'F 中 C、D'、F 的 y 坐标均为 1，故该平面为 y = 1。向量 A'B = (5/2,0,-√3/2)，其 y 分量为 0，故 A'B ∥ 平面 CD'F。

（2）平面 EFD'A' 的一个法向量为 n₁ = (√3/2,0,1/2)。平面 BCD' 的一个法向量可取 n₂ = (√3/2,√3/2,3/2)。

cos θ = |n₁·n₂| / (|n₁||n₂|) = √15 / 5，所以所求二面角的正弦值为 √(1 - 15/25) = 2√5 / 5。

18.（17 分）

已知函数 f(x) = ln(1 + x) - x + 1/2 x² - kx³，其中 0 < k < 1/3。

（1）证明：f(x) 在区间 (0,+∞) 存在唯一的极值点和唯一的零点；

（2）设 x₁、x₂ 分别为 f(x) 在区间 (0,+∞) 的极值点和零点。

（i）设函数 g(t) = f(x₁ + t) - f(x₁ - t)，证明：g(t) 在区间 (0,x₁) 单调递减；

（ii）比较 2x₁ 与 x₂ 的大小，并证明你的结论。

答案：（1）唯一极值点 x₁ = 1/(3k) - 1，且存在唯一正零点 x₂；（2）（i）g(t) 在 (0,x₁) 单调递减；（ii）2x₁ > x₂。

解：（1）f'(x) = 1/(1 + x) - 1 + x - 3kx² = x²[1/(1 + x) - 3k]。

令 f'(x) = 0，得 x₁ = 1/(3k) - 1 > 0。f'(x) 在 (0,x₁) 为正，在 (x₁,+∞) 为负，故唯一极值点为 x₁，且为极大值点。

又 f(0) = 0，x → 0⁺ 时 f(x) ~ (1/3 - k)x³ > 0，而 x → +∞ 时 f(x) → -∞。结合先增后减，得 (0,+∞) 上存在唯一零点 x₂。

（2）（i）由 3k = 1/(1 + x₁)，得 f'(x) = x²(x₁ - x) / [(1 + x)(1 + x₁)]。

g'(t) = f'(x₁ + t) + f'(x₁ - t)

= t/(1 + x₁) · [(x₁ - t)²/(1 + x₁ - t) - (x₁ + t)²/(1 + x₁ + t)]。

函数 h(x) = x²/(1 + x) 在 x > 0 上递增，故 g'(t) < 0，g(t) 在 (0,x₁) 单调递减。

（ii）由（i）知 t ∈ (0,x₁) 时 g(t) < g(0) = 0。令 t → x₁⁻，得 f(2x₁) ≤ f(0) = 0，且事实上 f(2x₁) < 0。

因为 f 在 (x₁,+∞) 单调递减，且 x₂ 是唯一正零点，所以 x₁ < x₂ < 2x₁，即 2x₁ > x₂。

19.（17 分）

甲、乙两人进行乒乓球练习，每个球胜者得 1 分，负者得 0 分。设每个球甲胜的概率为 p（1/2 < p < 1），乙胜的概率为 q，p + q = 1，且各球的胜负相互独立。对正整数 k ≥ 2，记 pₖ 为打完 k 个球后甲比乙至少多得 2 分的概率，qₖ 为打完 k 个球后乙比甲至少多得 2 分的概率。

（1）求 p₃，p₄（用 p 表示）；

（2）若 (p₄ - p₃) / (q₄ - q₃) = 4，求 p；

（3）证明：对任意正整数 m，p₂ₘ₊₁ - q₂ₘ₊₁ < p₂ₘ - q₂ₘ < p₂ₘ₊₂ - q₂ₘ₊₂。

答案：（1）p₃ = p³，p₄ = 4p³q + p⁴ = p³(4 - 3p)；（2）p = 2/3；（3）见证明。

解：（1）打完 3 个球甲至少多 2 分，只能甲全胜，所以 p₃ = p³。打完 4 个球甲至少多 2 分，即甲至少胜 3 球，所以 p₄ = C₄³p³q + p⁴ = 4p³q + p⁴。

（2）同理 q₃ = q³，q₄ = 4q³p + q⁴。于是 p₄ - p₃ = 3p³q，q₄ - q₃ = 3pq³。

故 (p₄ - p₃)/(q₄ - q₃) = p²/q² = 4。又 p + q = 1，p > q，得 p/q = 2，所以 p = 2/3。

（3）设打完 2m 个球后甲胜 r 球的概率为 Pᵣ。则 p₂ₘ = P(r ≥ m + 1)，q₂ₘ = P(r ≤ m - 1)。

由再打一球的边界项变化，p₂ₘ₊₁ = p₂ₘ - qPₘ₊₁，q₂ₘ₊₁ = q₂ₘ + qPₘ，因此 p₂ₘ₊₁ - q₂ₘ₊₁ = p₂ₘ - q₂ₘ - q(Pₘ + Pₘ₊₁) < p₂ₘ - q₂ₘ。

再打两球，有 p₂ₘ₊₂ - p₂ₘ = p²Pₘ - q²Pₘ₊₁，q₂ₘ₊₂ - q₂ₘ = q²Pₘ - p²Pₘ₋₁。

故 (p₂ₘ₊₂ - q₂ₘ₊₂) - (p₂ₘ - q₂ₘ) = p²Pₘ₋₁ + (p² - q²)Pₘ - q²Pₘ₊₁。

由 Pₘ₋₁ = [m/(m + 1)](q/p)Pₘ，Pₘ₊₁ = [m/(m + 1)](p/q)Pₘ，上式化为 (p - q)Pₘ > 0。

因此 p₂ₘ₊₁ - q₂ₘ₊₁ < p₂ₘ - q₂ₘ < p₂ₘ₊₂ - q₂ₘ₊₂。
